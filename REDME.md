## 📖 Sobre o Projeto

A **Plataforma de Gerenciamento de Voluntários** é um projeto solidário de aplicação web desenvolvida com foco em **cadastro e gerenciamento de voluntários**, com uma experiência visual moderna e temática cristã.

A aplicação permite que usuários se cadastrem e que administradores realizem operações completas de **CRUD**, com interface intuitiva e atualizações em tempo real.

## ✨ Funcionalidades

### 👤 Usuário

* Cadastro de voluntários
* Visualização de membros cadastrados
* Interface simples e intuitiva

### 🔐 Admin

* ✏️ Editar voluntários (via modal)
* ❌ Excluir voluntários
* 📋 Visualização completa dos dados
* ⚡ Atualização em tempo real (Firestore)

### 🎨 Interface

* Hero com vídeo de fundo
* Tema cristão com glow dourado ✨
* Header com efeito ao scroll (blur + transparência)
* Animações suaves (IntersectionObserver)
* Design responsivo
